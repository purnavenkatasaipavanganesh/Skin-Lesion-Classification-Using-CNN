# Automated-Classification-of-Skin-Lesions-Using-Neural-Networks
Automated Skin Lesion Classification using Deep Learning and ResNet50 for accurate detection of skin diseases from dermoscopic images. The system classifies seven types of skin lesions to support early diagnosis of melanoma and benign skin conditions using image-based analysis.

# Automated Skin Lesion Classification Using Deep Learning

## Overview
This project presents an automated skin lesion classification system using deep learning techniques. The main objective is to classify different types of skin lesions from dermoscopic images and assist in the early detection of skin cancer.

The model is developed using the ResNet50 architecture and trained on skin lesion datasets to achieve accurate classification results.

## Dataset
The dataset contains dermoscopic images of different skin lesion categories.

### Classes Used
1. Melanoma
2. Melanocytic Nevi
3. Basal Cell Carcinoma
4. Benign Keratosis
5. Actinic Keratoses
6. Vascular Lesions
7. Dermatofibroma
